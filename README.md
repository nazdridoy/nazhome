## Technologies Used

- Vanilla JavaScript (ES6+)
- Vite.js for building and development
- CSS3 with modern features
- Local Storage for data persistence
- OpenWeatherMap API for weather data
- Unsplash API for background images

- Chrome/Edge (latest)
